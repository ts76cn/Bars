# 子連れOK！仙台のバー・ダイニングバー
・飲食店APIを用いたGoogle Mapsへのマーカー表示

## Description　説明
・リクルートAPI利用
・子供を連れていける仙台駅・国分町・一番町周辺のバー・ダイニングバーのおすすめ１０店を抽出
・Google Maps上にマーカー表示しています。
・吹き出しには店舗名称とURLを表示しています。

## Requirement　要件
・PHP 7.2.21<br>
・リクルートWEBサービス グルメサーチAPI

## Usage　使用法
・map.phpが起動ファイルになります。
・$API_keyには、Google MapsのAPI keyを入力する必要があります。
